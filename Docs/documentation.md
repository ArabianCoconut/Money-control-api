# Function Descriptions for the functions in the package

### About

All the functions in this package returns JSON objects of the news from moneycontrol.com

-----------------

### get_news()

* This function returns the latest news from moneycontrol.com

### get_latest_news()

* This function returns the latest news from moneycontrol.com

### get_business_news()

* This function returns the latest business news from moneycontrol.com
